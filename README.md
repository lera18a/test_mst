# Flutter-фитнес-приложение с экраном подписки
---

##  Архитектура

-  Простая архитектура без использования state management библиотек 
-  Используется StatefulWidget для управления локальным UI-состоянием 
-  SharedPreferences для хранения статуса подписки
-  Настроена через Navigator.pushAndRemoveUntil

---

##  Структура проекта
lib/
├── main.dart # Точка входа, проверка подписки
├── global_page/
│ ├── global_page.dart # Экран со списком упражнений 
│ └── card_of_exercise.dart # Карточка упражнения
├── onbording/
│ └── onbording.dart # Онбординг-экран с переходом на paywall
├── paywall/
│ ├── paywall.dart # Экран выбора подписки
│ ├── paywall_item.dart # Карточка тарифа + логика оплаты
│ └── text_item.dart # Виджет бенефита
└── shared_preferences/
└── shared_preferences_service.dart # Сервис работы с локальным хранилищем

---

##  Возможности для улучшения

- Разделение на группы(категории): ноги, руки, спина, пресс и т.д
- Использовать Hero-анимацию при нажатии на карточку → переход на экран с описанием и видео 
- При выборе одного тарифа — блокировать кнопку другого
